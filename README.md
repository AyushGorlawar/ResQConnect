# ResQConnect
ResQConnect - Animal Rescue App For NGO

**ResQConnect** is a mobile-based animal rescue application designed to bridge the communication gap between citizens and animal rescue NGOs. The app enables users to submit accurate, location-based rescue requests, helping NGOs respond faster and more efficiently.
